Commandes Vim - Editeur de texte

Déplacement curseur :
  • h : vers la gauche
  • k : vers le haut
  • l : vers la droite
  • j : vers le bas
3 modes :
  • Commande
  • Insertion
  • Normal
echap : pour revenir en mode normal
  • :q ! : permet de sortir de vim (sans enregistrer les modifications)
  • :wq : permet d'enregistrer un fichier et de sortir
  • i : permet d'incruster du texte avant le curseur
  • A : permet d'ajouter du texte après le curseur
  • o : permet d'ouvrir une ligne sous le curseur et passer en mode insertion
  • O : permet d'ouvrir une ligne au-dessus du curseur et passer en mode insertion
  • a : permet d'insérer du texte après le curseur
  • x : permet d'effacer le caractère sous le curseur
  • dw : permet d'effacer un mot complet en se plaçant sur sa première lettre
  • d$ : permet d'effacer une ligne complète de l'endroit où l'on se positionne jusqu’à la fin de la ligne
  • dd : permet d'effacer une ligne (et p pour la replacer ailleurs après le curseur)
  • dNw : permet d'effacer N mots à partir du curseur
  • Ndd : permet d'effacer N lignes à partir du curseur
  • ce : permet d'effacer le mot à la suite du curseur et de passer en mode insertion pour changer la fin du mot
  • r suivi d'un lettre : permet de remplacer un caractère par la lettre sous le curseur
  • R : permet de remplacer plus d'un caractère
  • e : permet de se déplacer de mot en mot
  • Nw : déplace de N mot vers l'avant
  • Ne : déplace le curseur à la fin du N mot vers l'avant
  • NG : permet d'aller à la ligne N directement
  • G : permet de se déplacer à la fin du fichier
  • gg : permet de se déplacer au début du fichier
  • 0 : permet de se déplacer au début de la ligne
  • cNw : permet de se déplacer de N mots à partir du curseur
  • j$ : permet de déplacer le curseur à la fin de la ligne suivante
  • y : permet de copier du texte
  • p : permet de le coller
  • u : permet d'annuler la dernière modification
  • U : permet de récupérer toute une ligne
  • ctrl R : permet de revenir en arrière
  • ctrl G : permet d'afficher la position dans le fichier et son état
  • / suivi d'un fichier pour rechercher le fichier
  • :! suivi de la commande externe : permer d'exécuter une commande externe
  • :! ls : permet de sortir
  • :! dir
  • :w suivi du nom du fichier : permet d'enregistrer les changements faits au texte
  • :! rm suivi du nom du fichier : permet d'effacer le fichier
  • : help ou F1 : permet d'obtenir de l'aide (ouvre le texte dans la fenêtre)
  • ctrl w : permet de passer d'une fenêtre à l'autre
  • :q : permet de sortir
